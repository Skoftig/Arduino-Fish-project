# Arduino-Fish-project

This is the repository of files on my Arduino Magikarp project. 
Above you'll find my code, and a logbook I wrote through the duration of this project.

- 1x Arduino Uno
- 1x Servo motor
- 1x Piezo Buzzer
- 1x LDR sensor
